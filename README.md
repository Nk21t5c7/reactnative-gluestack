# React Native Gluestack Memo App

## Overview
This is a simple memo app created to learn and explore Gluestack UI.
It demonstrates:
- Displaying a list of memos
- Creating new memos
- Simple state management using useSyncExternalStore
- Usage of Gluestack UI components like VStack, Box, and FormControl

## Tech Stack
- React Native + Expo
- Gluestack UI v3
